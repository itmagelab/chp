# Технические характеристики Changan Hunter Plus

## Общая информация

| Параметр | Значение |
|----------|----------|
| Производство | с 2023 года |

| Кузов | двойная кабина (4 двери) |
| Платформа | Changan F70 |
| Количество мест | 5 |

## Двигатель

| Параметр | Значение |
|----------|----------|
| Тип | бензиновый, турбированный |
| Объём | 2.0 л (1998 см³) |
| Конфигурация | R4 (рядный, 4 цилиндра) |
| Мощность | 226 л.с. |
| Крутящий момент | 390 Н·м |
| Диапазон мощности (разные рынки) | 163–233 л.с. |
| Особенности | непосредственный впрыск, турбонаддув |

### Маркировка двигателя

- Точный код двигателя официально редко указывается в открытых источниках  
- Используется 2.0T семейства Changan (аналогичные моторы серии **JL486ZQ / JL486ZQ5**)

---

## Трансмиссия

| Параметр | Значение |
|----------|----------|
| Тип КПП | автоматическая |
| Количество передач | 8 |
| Привод | полный (подключаемый 4WD) |
| Раздатка | есть (с понижающей передачей) |

### Маркировка коробки

- Гидротрансформаторная АКПП  
- Конструктивно близка к коробкам класса ZF 8HP (китайские аналоги)  
- Точное обозначение зависит от рынка

## Важно знать

- Используется пружинная задняя подвеска вместо рессор — это улучшает комфорт  
- Полный привод с режимом **AUTO** удобен для повседневной езды  
- Высокий крутящий момент (~390 Н·м) делает автомобиль пригодным для бездорожья и перевозки грузов  
- Классическая рамная конструкция обеспечивает прочность и выносливость
